Global Data

Global data can be defined in .js or .json files in the _data directory. When defined in a .js file, it should export a function that returns the data. For example, the file _data/hockey.js could contain:

module.exports = () => [
  {city: 'Chicago', name: 'Blackhawks'},
  {city: 'St. Louis', name: 'Blues'}
];
